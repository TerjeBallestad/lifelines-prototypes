       ```bash
       npm install mobx@^6.15.0 mobx-react-lite@^4.1.1

       npm install -D @types/node
       ```

       ```javascript
       export default {
         plugins: {

         },
       };
       ```

    4. Replace src/index.css with Tailwind 4 + DaisyUI dark theme config:
       ```css
       @import "tailwindcss";

       @plugin "daisyui" {
         themes: dark --default;
         logs: false;
       }

       :root {
         color-scheme: dark;
       }
       ```

    5. Clean up Vite boilerplate:
       - Remove src/App.css (not needed with Tailwind)
       - Simplify src/App.tsx to minimal component:

         ```tsx
                 Lifelines Prototype

         export default App;
         ```

    6. Verify dev server starts: `npm run dev`

<task type="auto">
  <name>Task 2: Configure TypeScript strict mode and ESLint 9</name>
  <files>
    tsconfig.json
    tsconfig.app.json
    eslint.config.js
    .prettierrc
  </files>
  <action>
    1. Update tsconfig.json to extend strict settings. Add to compilerOptions:
       ```json
       {
         "compilerOptions": {
           "strict": true,
           "noImplicitAny": true,
           "strictNullChecks": true,
           "strictFunctionTypes": true,
           "strictPropertyInitialization": true,
           "noUncheckedIndexedAccess": true,
           "useUnknownInCatchVariables": true
         }
       }
       ```
       Note: Vite template may split configs. Apply strict settings to tsconfig.app.json if that's where compilerOptions live.

    2. Install ESLint 9 flat config dependencies (if not already present):
       ```bash
       npm install -D eslint@^9 prettier@^3 @typescript-eslint/parser @typescript-eslint/eslint-plugin
       npm install -D eslint-plugin-react-hooks eslint-plugin-react-refresh eslint-config-prettier
       ```

    3. Replace eslint.config.js with flat config (from RESEARCH.md):
       ```javascript
       import js from '@eslint/js';
       import tseslint from 'typescript-eslint';
       import reactHooks from 'eslint-plugin-react-hooks';
       import reactRefresh from 'eslint-plugin-react-refresh';
       import eslintConfigPrettier from 'eslint-config-prettier';

       export default tseslint.config(
         { ignores: ['dist'] },
         {
           extends: [
             js.configs.recommended,
             ...tseslint.configs.strict,
             ...tseslint.configs.stylistic,
           ],
           files: ['**/*.{ts,tsx}'],
           plugins: {
             'react-hooks': reactHooks,
             'react-refresh': reactRefresh,
           },
           rules: {
             ...reactHooks.configs.recommended.rules,
             'react-refresh/only-export-components': [
               'warn',
               { allowConstantExport: true },
             ],
           },
         },
         eslintConfigPrettier,
       );
       ```

    4. Create .prettierrc:
       ```json
       {
         "semi": true,
         "singleQuote": true,
         "trailingComma": "es5",
         "tabWidth": 2,
         "printWidth": 80
       }
       ```

    5. Run lint check: `npm run lint`
    6. Run type check: `npx tsc --noEmit`
  </action>
  <verify>
    - `npm run lint` passes (or shows only warnings, no errors)
    - `npx tsc --noEmit` passes without type errors
    - TypeScript catches `any` types if accidentally used
  </verify>
  <done>
    TypeScript strict mode active, ESLint 9 flat config working, Prettier configured.
  </done>
</task>
